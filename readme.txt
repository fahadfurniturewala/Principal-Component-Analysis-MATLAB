Instructions to run the function pca_power in MATLAB.
The name of the function is pca_power.
In the MATLAB command window enter the following commands.


1.type  " pca_power pendigits_training.txt pendigits_test.txt '1' '10' " in the command window in Matlab
2.Press enter



The first argument is the training file name which can be changed as and when needed.
The second argument is the testing file name which can be changed as and when needed.
The third argument is the number of dimensions the output matrix should have.
Th fourth argument is the number of iterations.

The other possible running combinatins are:

pca_power satellite_training.txt satellite_test.txt '2' '20'
pca_power yeast_training.txt yeast_test.txt '3' '30'
